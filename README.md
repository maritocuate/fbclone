# Facebook Clone

Building facebook application clone using `React`.

### 🛠 Tech Used
- **Next 12**
- **Tailwindcss 3**
- **ESLint 8**

### ⭐ How to run locally
- `npm install`
- `npm start`

### ✅ Build for production
- `npm build` -> Will output a build folder

### 🚥 Linting
- `npm lint` -> Will lint all code under `src` folder. Uses **JavaScript Standard Style**
